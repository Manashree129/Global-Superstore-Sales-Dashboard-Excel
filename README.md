# 📊 Global Superstore Sales Dashboard (Excel)

## 📌 Project Overview

This project is an interactive Sales Dashboard built in Microsoft Excel using the Global Superstore dataset.

The dashboard enables users to analyze sales performance dynamically using Pivot Tables, Pivot Charts, KPI Cards, Slicers, and Timeline.

---

## 🎯 Objectives

- Clean and validate raw sales data
- Build interactive KPI cards
- Analyze sales trends
- Compare markets and categories
- Create business insights using Excel

---

## 🛠 Tools & Features Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Timeline
- Helper Columns
- Conditional Formatting
- Data Cleaning
- Data Validation

---

## 📊 Dashboard KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Quantity Sold
- Profit Margin
- Average Delivery Days
- Average Discount

---

## 📈 Dashboard Visualizations

- Sales Trend
- Sales by Category
- Sales by Market
- Sales by Sub-Category

---

## 📋 Data Quality Report

The project includes:

- Missing Value Analysis
- Distinct Count Validation
- Statistical Summary
- Data Cleaning Checks

---

## 🔍 Key Business Insights

- Technology generated the highest sales.
- APAC was the top-performing market.
- Phones were the highest-selling sub-category.
- Sales increased steadily from 2011 to 2014.
- Standard Class was the most-used shipping mode.
